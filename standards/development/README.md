# Development Standards (Canonical)

This directory defines the canonical engineering standards for the InterDead project ecosystem.

## Scope

These standards apply to any repository that contains software, build tooling, or operational automation. Individual repositories may add repo-specific notes, but must not redefine the canonical rules.
